# 色愁久久久人愁久下一句怎么接

自己以后一定要小心再小心，必须要明白高人的任何一个暗示，争取让高人百分百满意。


第二十三章 言出法随
　　白无尘更是苦笑连连，心中暗叹。
　　自己之前怎么就没能领悟高人的意思呢？居然没有提前给高人订好酒楼，反而让林清云抢了先，高人这是在提点我啊！
　　赵老头注意到白无尘时不时就长吁短叹，不由得更为的慎重，仔细听着李念凡讲的故事。
　　此时，刚好讲到菩提祖师给孙悟空传道。
　　“祖师道：“你今有缘，我亦喜说。既识得盘中暗谜，你近前来，仔细听之，当传与你长生之妙道也。”悟空叩头谢了，洗耳用心，跪于榻下……”
　　有缘？
　　喜说？
　　长生之妙道？
　　林清云、白无尘、赵老头都是全身一震，瞳孔深处带着一丝激动和震撼。
　　身躯都在微微的颤抖。
　　真的会讲长生之妙道吗？
　　他们心中笃定，这是高人借助菩提祖师的口来传授自己，暗示自己跟他有缘。
　　尤其是赵老头，他行将就木，陡然听到长生之妙道，不由得连呼吸都屏住了。
　　孟君良握笔的手也是紧了紧，抿着嘴巴，生怕漏听一个字。
　　“祖师云：“显密圆通真妙诀，惜修生命无他说。都来总是精气神，谨固牢藏休漏泄。休漏泄，体中藏，汝受吾传道自昌……”
　　一重重口诀从李念凡的嘴里传出，如同暮鼓晨钟，伴随着大道之音传入所与人的耳中，有些人眼神微微一凝，若有所悟，有些人则是面色如常，毫无感觉。
　　白无尘等人身子已然僵直，脑海中不断的传来轰轰之声。
　　大道！
　　真的是长生大道！
　　他们头皮发麻，全身鸡皮疙瘩都掉了一地。
　　这些句子晦涩难懂，他们听得云里雾里，只得连忙牢牢记在心底。
　　轰隆！
　　一层厚厚的乌云毫无征兆的笼罩在落仙城的上空，闪电如同银蛇一般在云层中游走，将整个落仙城都照成了银色。
　　哗啦啦！
